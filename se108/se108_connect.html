SE108 Configuration Utility

The SE108 is a submersible datalogging and control system with eight wet matable prewired connectors used for the purpose of connecting sensors to power and communications

Connecting to the SE108 - Serial

  1. Ensure that the system, specifically the datalogger, is powered up.
  2. Connect your computer to the datalogger's RS-232 port.
  3. Select the correct Communication Port in the left panel.
  4. Select the Connect button

Connecting to the SE108 - TCP/IP

If the datalogger within the SE108 has an Internet/IP network interface, it may be possible to connect remotely using TCP/IP.
  1. Ensure that the system, specifically the datalogger, is powered up.
  2. Ensure that your computer can reach the datalogger via an IP connection.
  3. Check the Use IP Connection selection in the left panel.
  4. Enter the datalogger's IP address and TCP port number (IP:Port) into Communication Port.
  5. Enter the correct PakBus/TCP Password and PakBus Encryption Key if used.
  6. Select the Connect button
